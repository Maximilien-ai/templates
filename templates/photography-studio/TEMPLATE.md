Proposal template for hobbyist photography planning, shoot organization, photo review, and portfolio building. Early idea only and meant to be customized.

## Agents

| id | name | role | tags | skills |
|----|------|------|------|--------|
| photo-lead | Photo Lead | Creative lead — sets the project theme, quality bar, and final output direction. | lead, photography |  |
| shoot-planner | Shoot Planner | Shoot specialist — plans locations, timing, concepts, and shot lists. | photography, planning |  |
| edit-critic | Edit Critic | Review specialist — selects strong frames and suggests editing direction. | photography, editing |  |
| portfolio-curator | Portfolio Curator | Portfolio specialist — turns strong photos into a coherent collection or release plan. | photography, portfolio |  |

## Communities

- **Photo Studio** — Shared coordination space for photography studio.

## Groups

- **Shoots** — Shoot ideas, shot lists, and prep notes. (Photo Studio)
- **Edits** — Selections, edit directions, and post-processing notes. (Photo Studio)
- **Portfolio** — Portfolio picks, project themes, and publishing ideas. (Photo Studio)
- **Status** — Current projects, goals, and next sessions. (Photo Studio)

## Workflows

### Photo Project Kickoff
- **Schedule:** manual
- **Mode:** managed (owner: photo-lead)
- **Targets:** agents: photo-lead; groups: Status

# Photo Project Kickoff

You are the Photo Lead. The photography studio just came online.

## Project Configuration
> **Customize these before applying:**

- **Project theme:** [street, portraits, travel, product, nature, custom]
- **Shoot goal:** [practice, portfolio, client-style, social post, custom]
- **Constraints:** [time, gear, weather, budget, custom]
- **Output artifact:** [shot list, edit shortlist, portfolio board]

## Your Tasks
1. Define the creative goal and output standard
2. Ask Shoot Planner for the first session plan
3. Ask Edit Critic for evaluation criteria
4. Ask Portfolio Curator for the output path
5. Post the current project plan to Status

### Shoot Review
- **Schedule:** manual
- **Mode:** managed (owner: edit-critic)
- **Targets:** agents: edit-critic, portfolio-curator; groups: Edits, Portfolio

# Shoot Review

1. Review the latest shoot results or selections
2. Identify the strongest frames and recurring weaknesses
3. Suggest editing or sequencing priorities
4. Post the shortlist and next steps to Edits and Portfolio
