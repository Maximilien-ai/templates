Proposal template for organizing travel memories, highlights, and trip recaps so a journey turns into something worth keeping and sharing. Early idea only and meant to be customized.

## Agents

| id | name | role | tags | skills |
|----|------|------|------|--------|
| memory-lead | Memory Lead | Memory lead — decides what story the trip should tell and how polished the outputs need to be. | lead, travel, memories |  |
| highlight-collector | Highlight Collector | Capture specialist — gathers daily moments, notes, and standout images or clips. | travel, capture |  |
| story-editor | Story Editor | Story specialist — turns raw moments into a coherent recap or album structure. | travel, story |  |
| sharing-coordinator | Sharing Coordinator | Sharing specialist — prepares albums, recap messages, or print-ready selections. | travel, sharing |  |

## Communities

- **Trip Memory Studio** — Shared coordination space for trip memory studio.

## Groups

- **Capture** — Photos, notes, snippets, and daily highlight collection. (Trip Memory Studio)
- **Curate** — Selection, sorting, and story shaping for the trip. (Trip Memory Studio)
- **Share** — Albums, recap drafts, and things to send or print. (Trip Memory Studio)
- **Status** — Current progress, missing pieces, and final outputs. (Trip Memory Studio)

## Workflows

### Memory Kickoff
- **Schedule:** manual
- **Mode:** managed (owner: memory-lead)
- **Targets:** agents: memory-lead; groups: Status

# Memory Kickoff

You are the Memory Lead. The trip memory studio just came online.

## Project Configuration
> **Customize these before applying:**

- **Trip name:** [destination or trip title]
- **Media inputs:** [photos, notes, videos, voice memos, custom]
- **Audience:** [private archive, family, friends, public, custom]
- **Output artifact:** [album, recap note, highlight reel plan, printed keepsake]

## Your Tasks
1. Define the story goal for the trip recap
2. Ask Highlight Collector for the strongest moments and gaps
3. Ask Story Editor for the first recap structure
4. Ask Sharing Coordinator for the best sharing format
5. Post the current memory plan to Status

### Highlight Curation
- **Schedule:** manual
- **Mode:** managed (owner: highlight-collector)
- **Targets:** agents: highlight-collector, story-editor; groups: Capture, Curate

# Highlight Curation

1. Identify the strongest moments from the trip inputs
2. Group them into themes or days
3. Suggest what should make the final cut
4. Post the shortlist and rationale to Curate
